บทความที่1ซ HTML5 คืออะไร?

HTML5 คือเวอร์ชั่นล่าสุดของภาษา HTML ที่มาพร้อมคุณสมบัติใหม่มากมาย เช่น กรรองรับมัลติมีเดีย การจัดเก็บข้อมูล เเละเเท็กเชิงความหมาย

บทความที่ 2ซ ประโยชน์ของ Samantic tags

การใช้ semantic Tags เช่น header,nav,section,article,footer ทำให้โค้ด HTML มีโครงสร้างที่ชัดเจนเเละอ่านง่ายขึ้น ซึ่งเป็นประโยชน์ต่อ SEO เเละการเข้าถึง
